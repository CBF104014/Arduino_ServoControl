# Arduino Servo Control 伺服馬達控制車  
  
#### 使用ESP8266(WIFI)控制兩個SG90伺服馬達，並且通過HTTPS向遠端伺服器發送請求，根據伺服器的回應來控制車子。  
□ ESP8266開發版 * 1  
□ SG90伺服馬達(360度) * 2  
□ 杜邦線母對母10cm * 6  
□ 汽車本體+瓶蓋+行動電源 * 1
